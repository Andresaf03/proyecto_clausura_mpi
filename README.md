# Bolsa de Palabras con MPI

## Descripción

Este es el proyecto de clausura para la materia de Cómputo Paralelo y en la Nube que consiste en implementar una Bolsa de Palabras con MPI.

Se implementa una versión serial y una paralela para comparar el tiempo de ejecución y obtener un speed up de al menos 1.2x. Adicionalmente se cumple con el uso responsable y ético de la IA Generativa de acuerdo a los lineamientos de IEEE y Elsevier.
